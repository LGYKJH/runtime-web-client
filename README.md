## 시작하기
1. 저장소 클론
    ```bash
    git clone https://github.com/LGYKJH/runtime-web-client.git
    ```

2. 패키지 설치
    ```bash
    npm install
    # 또는
    yarn install
    ```

3. 개발 서버 실행
    ```bash
    npm run dev
    # 또는
    yarn dev
    ```

4. 프로젝트 빌드
    ```bash
    npm run build
    # 또는
    yarn build
    ```





This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
